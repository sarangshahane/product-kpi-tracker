# Product KPI Tracker

A WordPress plugin to track and analyze key performance indicators for your WooCommerce products.

## Features
- Track product performance metrics
- Generate KPI reports
- Monitor sales trends
- Analyze product data
- Dashboard with key metrics visualization

## Requirements
- WordPress 5.6+
- WooCommerce 5.0+
- PHP 7.4+

## Dashboard Features
The dashboard provides an at-a-glance view of your store's performance with the following KPIs:
- Net Revenue
- Monthly Recurring Revenue (MRR)
- Average Revenue Per Subscription (ARPS)
- Average Order Value (AOV)
- Churn Rate
- Refund Rate
- Cart Abandonment Rate

## Installation
1. Upload the plugin files to the `/wp-content/plugins/product-kpi-tracker` directory
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Navigate to WooCommerce → Product KPIs Tracker to view your dashboard

## Development
To set up the development environment:

1. Clone the repository
2. Run `npm install` to install dependencies
3. Run `npm start` to start the development server
4. Run `npm run build` to build for production
